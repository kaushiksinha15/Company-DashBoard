📊 Company Registration & Management System

📌 Overview

This project is a database-driven application designed to register companies and manage their essential details such as contact number and address. It provides efficient data storage, retrieval, and update functionalities while ensuring data integrity and consistency.

⸻

🚀 Features
	•	➕ Add new company records
	•	📝 Update company details (contact number, address, etc.)
	•	🔍 View and search company information
	•	🗑️ Delete company records
	•	✅ Data validation for reliable inputs

⸻

🛠️ Tech Stack
	•	Programming Language: Java
	•	Database: MySQL / SQL
	•	Connectivity: JDBC
	•	IDE: VS Code / IntelliJ
